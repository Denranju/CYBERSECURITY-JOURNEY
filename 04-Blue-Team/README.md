# 📚 Topics Covered

## Blue Team

Blue Team refers to cybersecurity professionals responsible for defending an organization's systems, networks and data against cyber attacks.

Their responsibilities include:

- Monitoring systems.
- Detecting threats.
- Investigating alerts.
- Responding to incidents.
- Improving security.

---

## Security Monitoring

Security monitoring involves continuously observing systems and networks for suspicious activity.

Monitoring helps identify attacks before they cause significant damage.

---

## Logging

Logs record activities performed by users, applications and operating systems.

Logs are valuable for:

- Troubleshooting
- Incident investigations
- Threat detection
- Compliance

---

## Incident Response

Incident Response is the structured process used to manage cybersecurity incidents.

Typical phases include:

1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

---

## Security Hardening

Security hardening reduces the attack surface by removing unnecessary services and improving system security.

Examples include:

- Applying software updates.
- Disabling unused services.
- Enforcing strong passwords.
- Configuring firewalls.
- Restricting user permissions.

---

## Authentication

Authentication verifies the identity of users before access is granted.

Common methods include:

- Passwords
- Multi-Factor Authentication (MFA)
- Biometrics
- Security Keys

---

## Authorization

Authorization determines what an authenticated user is allowed to access.

Applying the Principle of Least Privilege reduces security risks.

---

## Backups

Regular backups help organizations recover data after hardware failures, ransomware attacks or accidental deletion.

Backups are an important part of disaster recovery planning.

---

## Network Security

Network security protects devices and communications using technologies such as:

- Firewalls
- VPNs
- Intrusion Detection Systems
- Intrusion Prevention Systems

---

## Linux Security

Linux security includes:

- Managing file permissions.
- Using SSH securely.
- Limiting privileged access.
- Monitoring logs.
- Applying updates.

---

## Windows Security

Windows security includes:

- User Account Control (UAC)
- Windows Defender
- Windows Firewall
- Windows Event Logs
- Security Updates

---

## Threats

Common security threats include:

- Malware
- Phishing
- Ransomware
- Insider Threats
- Social Engineering
- Denial-of-Service Attacks
